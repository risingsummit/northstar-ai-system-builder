# Northstar AI System Builder

A standalone browser-based AI system planning workspace for portfolio demos and LinkedIn showcasing.

## Live Demo

Once this repo is published with GitHub Pages, the live URL will be:

```text
https://risingsummit.github.io/northstar-ai-system-builder/
```

## What It Does

Northstar AI System Builder turns a product idea into a practical AI system plan:

- Architecture map for experience, data, orchestration, evaluation, privacy, and operations
- Model strategy for routing, retrieval, classification, fallbacks, and tool control
- Guardrails for grounding, abuse filtering, audit logging, and sensitive data handling
- Launch roadmap from prototype to pilot to scale
- Operating metrics for readiness, latency, monthly cost, answer quality, safety, and automation fit

## Run Locally

Open `index.html` in a browser.

No build step, server, API key, or package install is required.

## Suggested GitHub Pages Setup

1. Create a new GitHub repo named `northstar-ai-system-builder`.
2. Upload `index.html`, `styles.css`, `app.js`, `README.md`, and the `assets` folder.
3. Go to repo settings, then Pages.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save.
